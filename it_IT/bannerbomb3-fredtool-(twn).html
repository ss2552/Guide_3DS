    BannerBomb3 + Fredtool (TWN)

    Se hai bisogno di aiuto, chiedi pure su Nintendo Homebrew su Discord (in inglese).
    Se hai gradito questa guida, accettiamo Donazioni.

    Lettura necessaria

    Per eseguire un dump dei DSiWare di sistema, approfittiamo di una falla di sicurezza nella sezione Gestione Nintendo DSiWare dell’applicazione Impostazioni della console.

    Per realizzare l’obiettivo, utilizziamo la chiave di cifratura della tua console (movable.sed) per generare un backup DSiWare che induce la console a creare un dump dell’applicazione Impostazioni Internet DSi nella directory principale della scheda SD.

    Una volta ottenuto un backup DSiWare, si può initiettare nella “Modalità download” un titolo DSiWare utilizzabile per installare un custom firmware.

    Queste istruzioni sono SOLO per le console taiwanesi (indicate da una T alla fine della versione di sistema, ad esempio 11.16.0-39T)! Se la tua console proviene da qualsiasi altra regione, scegli l’altro metodo.

    Cosa serve

    • Il tuo file movable.sed dalla sezione Seedminer
    • L’ultima versione di Luma3DS (il file .zip di Luma3DS)
    • L’ultima versione di b9stool (download diretto)
    • L’ultima versione di Frogminer_save (Frogminer_save.zip)

    Sezione I - Controllo del CFW

    Come precauzione aggiuntiva, controlleremo anche la presenza di un custom firmware. È necessario poiché se un custom firmware fosse già installato, questo metodo rischierebbe di brickare la console (rendendola inutilizzabile in assenza di metodi di ripristino come ntrboot).

    1. Spegni la tua console
    2. Premi il pulsante (Select)
    3. Accendi la tua console continuando a tenere premuto il pulsante (Select)
    4. Se il controllo è andato a buon fine, verrà avviato il menu HOME e potrai procedere con questa guida
    5. Spegni la tua console

    Se vedi un menu di configurazione o la console si spegne, hai già il CFW, e proseguire seguendo queste istruzioni può BRICKARE la console! Procedi con la Verifica del CFW per aggiornare il tuo CFW esistente.

    Sezione II - Preparazione

    1. Spegni la tua console
    2. Inserisci la scheda SD nel tuo computer
    3. Copia i file boot.firm e boot.3dsx dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
      • La directory principale della scheda SD è la cartella della tua scheda SD in cui è possibile visualizzare la cartella Nintendo 3DS, ma non il suo interno
    4. Copia il file boot.nds (B9STool) dall’archivio .zip di b9sTool nella directory principale della tua scheda SD
    5. Copia la cartella private presente nell’archivio .zip di Frogminer_save nella directory principale della tua scheda SD
    6. Tieni la scheda SD nel tuo computer - ci sono altre cose da fare nella prossima sezione

    Sezione III - BannerBomb3

    1. Apri il sito DSIHaxInjector V2 sul tuo computer
    2. Sotto il campo “Username” inserisci un nome alfanumerico qualunque (senza spazi o caratteri speciali)
    3. Sotto il campo “MovableSed”, carica il tuo file movable.sed cliccando sull’opzione “Browse…”
    4. Clicca “Build”
      • Attendi qualche secondo che termini il processo di compilazione
    5. Nella sezione “Build History” a sinistra, inserisci l’Username nel campo “Filter Builds”
    6. Clicca sul primo risultato della ricerca
      • Dovrebbe avere la data più recente
    7. Clicca il link “output_(name).zip”
    8. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1>
      • <ID0> è il nome a 32 lettere della cartella che hai copiato in Seedminer
      • <ID1> è una cartella dal nome lungo 32 caratteri all’interno di <ID0>
      • Se hai più cartelle <ID1>, segui queste istruzioni
    9. Crea una cartella chiamata Nintendo DSiWare all’interno di <ID1>
      • Se hai già la cartella e sono già presenti dei file di backup DSiWare (<id-a-8-caratteri>.bin) all’interno, copiali sul PC e rimuovili dalla scheda SD
    10. Apri il file .zip output_(nome) e naviga su China_Taiwan -> output_setup
    11. Copia tutti i sei file .bin nella cartella Nintendo DSiWare
    12. Reinserisci la scheda SD nella tua console
    13. Accendi la tua console
    14. Avvia le Impostazioni di sistema sulla tua console
    15. Entra in Gestione dati -> DSiWare -> Scheda SD -> (pagina 2)
    16. Prendi nota del numero sul titolo DSiWare nella seconda pagina (sarà un numero compreso tra 1 e 6)
    17. Spegni la tua console
    18. Inserisci la scheda SD nel tuo computer
    19. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
    20. Apri il file .zip output_(nome) e naviga su China_Taiwan -> output_hax
    21. Copia il file .bin con lo stesso numero di quello nella seconda pagina del menu “Gestione Nintendo DSiWare” nella cartella Nintendo DSiWare, e sovrascrivi quando richiesto
    22. Accendi la tua console
    23. Avvia le Impostazioni di sistema sulla tua console
    24. Entra in Gestione dati -> DSiWare -> Scheda SD (image)
    25. Attendi
      • Dovrebbero succedere in ordine le seguenti cose: Una barra di caricamento, un breve blocco, lo schermo inferiore viola, l’arresto della musica, infine la console mostrerà un errore e si riavvierà
      • Se non hai visto lo schermo viola, qualcosa è andato storto
    26. Ora dovresti avere il file 42383821.bin nella directory principale della tua scheda SD
    27. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
    28. Elimina tutti i file .bin dalla cartella Nintendo DSiWare

    Sezione IV - Fredtool

    1. Apri il sito DSIHaxInjector_new sul tuo computer
    2. Sotto il campo “Username” inserisci un nome alfanumerico qualunque (senza spazi o caratteri speciali)
      • Ti conviene inserire un nome diverso per differenziarlo dall’output di BannerBomb3
    3. Sul campo “DSiBin”, carica il tuo file 42383821.bin usando la prima opzione “Browse…”
    4. Sul campo “MovableSed”, carica il tuo file movable.sed usando la seconda opzione “Browse…”
    5. Sul campo “InjectionTarget”, imposta l’obiettivo dell’iniezione su DSdownloadplay (NON memorypit)
    6. Clicca “Build”
      • Attendi qualche secondo che termini il processo di compilazione
    7. Nella sezione “Build History” a sinistra, inserisci l’Username nel campo “Filter Builds”
    8. Clicca sul primo risultato della ricerca
      • Dovrebbe avere la data più recente
    9. Clicca il link “output_(name).zip”
    10. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
    11. Copia il file 484E4441.bin dalla cartella hax dall’archivio DSiWare appena scaricato (output_(name).zip) nella cartella Nintendo DSiWare
    12. Reinserisci la scheda SD nella tua console
    13. Accendi la tua console
    14. Avvia le Impostazioni di sistema sulla tua console
    15. Entra in Gestione dati -> DSiWare
    16. Nella sezione “Scheda SD”, scegli il titolo “Haxxxxxxxxx!”
    17. Seleziona “Copia”, quindi “OK”
    18. Esci da Impostazioni della console
    19. Avvia “Modalità download” sulla tua console (l’icona arancione con un 3DS)
    20. Seleziona “Nintendo DS”
    21. Se l’exploit ha avuto successo, la tua console avrà caricato la versione JPN di Flipnote Studio

    Sezione V - Exploit Flipnote

    Se desideri una guida visiva per questa parte, è disponibile qui.

    In questa sezione, eseguirai una serie passaggi molto specifici da fare in Flipnote Studio che, se fatti correttamente, farà partire b9sTool, l’installer di boot9strap (custom firmware).

    1. Completa la configurazione iniziale del gioco finché non avrai raggiunto il menu principale
      • Seleziona l’opzione di sinistra quando richiesto durante la configurazione
      • Se riscontri un problema durante questa sezione, cerca in questa guida il tuo problema
    2. Con il touch screen, seleziona il riquadro grande a sinistra, poi il riquadro con l’icona di una scheda SD
    3. Al termine del caricamento del menu, scegli l’icona col volto, poi premi l’icona in basso a destra per proseguire
    4. Premi (X) o (SU) sul D-Pad a seconda di cosa sia visualizzato sullo schermo superiore
    5. Seleziona il secondo pulsante in cima, con l’icona di una pellicola
    6. Scorri a destra fino a visualizzare la bobina “3/3”
    7. Seleziona il terzo riquadro con la lettera “A” all’interno
    8. Scorri a sinistra fino a visualizzare la bobina “1/3”
    9. Seleziona il quarto riquadro con la lettera “A” all’interno
    10. Se l’exploit ha funzionato, la tua console avvierà b9sTool
    11. Usando il D-Pad, vai su “Install boot9strap”
      • Se salti questo passaggio, la console si riavvierà nel menu HOME senza aver installato boot9strap, e sarà necessario riaprire l’applicazione Modalità download DS e ricominciare dall’inizio di questa sezione
    12. Premi (A), quindi premi START e SELECT contemporaneamente per avviare il processo
    13. Una volta finito e nello schermo inferiore c’è scritto “done.”, esci da b9sTool e spegni la console
      • Potresti dover forzare lo spegnimento tenendo premuto il pulsante di accensione
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
      • Se vedi la schermata di Luma, spegni la console e prosegui alla prossima sezione

    Sezione VI - Configurazione di Luma3DS

    1. Premi (Select) e accendi la tua console continuando a tenerlo premuto
    2. La tua console dovrebbe aver avviato il menu di configurazione di Luma3DS
      • Il menu di configurazione di Luma3DS contiene le impostazioni per il custom firmware Luma3DS. Molte di queste impostazioni possono essere utili per la personalizzazione o per debug
      • Ai fini di questa guida, lascia queste opzioni nelle impostazioni predefinite (non spuntare o deselezionare nulla)
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
    3. Premi (Start) per salvare e riavviare il 3DS

    Sezione VII - Ripristino della Modalità Download DS

    1. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
    2. Copia il file 484E4441.bin dalla cartella clean dall’archivio DSiWare appena scaricato (output_(name).zip) nella cartella Nintendo DSiWare`
    3. Reinserisci la scheda SD nella tua console
    4. Accendi la tua console
    5. Avvia le Impostazioni di sistema sulla tua console
    6. Entra in Gestione dati -> DSiWare
    7. Nella sezione “Scheda SD”, scegli il titolo “Haxxxxxxxxx!”
    8. Seleziona “Copia”, quindi “OK”
    9. Esci da Impostazioni della console
    10. Spegni la tua console
