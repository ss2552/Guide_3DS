    Homebrew Launcher (ninjhax2-dx)

    Se hai bisogno di aiuto, chiedi pure su Nintendo Homebrew su Discord (in inglese).
    Se hai gradito questa guida, accettiamo Donazioni.

    Indice

    • Cosa serve
    • Istruzioni
      • Sezione I - Preparazione
      • Sezione II - ninjhax2-dx
      • Sezione III - Verifica Pulsanti Hardware
    • Passaggio successivo: Scelta di un exploit
    Dettagli Tecnici (opzionale)

    Ninjhax2 è un exploit per il gioco “Cubic Ninja” realizzato da smealum. Questa pagina utilizza una versione aggiornata di ninjhax di zoogie chiamata ninjhax2-dx, che supporta la versione 11.17.0.

    Questo metodo richiede una fotocamera 3DS funzionante.

    Questo processo sovrascriverà il tuo file di salvataggio per Cubic Ninja, se ne hai uno.

    Cosa serve

    • Il gioco “Cubic Ninja”
    • L’ultima versione di ninjhax2-dx
    • L’ultima versione di Luma3DS (il file .zip di Luma3DS)

    Istruzioni

    Sezione I - Preparazione

    1. Spegni la tua console
    2. Inserisci la scheda SD nel tuo computer
    3. Copia il file Launcher.dat dalla cartella appropriata per la console dall’archivio .zip di ninjhax2-dx nella directory principale della tua scheda SD
      • La directory principale della scheda SD è la cartella della tua scheda SD in cui è possibile visualizzare la cartella Nintendo 3DS, ma non il suo interno
    4. Copia i file boot.firm e boot.3dsx dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD

    Sezione II - ninjhax2-dx

    1. Reinserisci la scheda SD nella tua console
    2. Accendi la tua console
    3. Lancia “Cubic Ninja” la prima volta e seleziona “Crea”
      • Se richiesto, elimina tutto i dati
    4. Seleziona “QR Code”
    5. Seleziona “Scansiona QR Code”
    6. Sul tuo PC, apri il browser e vai al sito web di ninjhax2-dx
    7. Seleziona la regione e l’hardware della tua console
    8. Sulla tua console, scansiona il codice QR
      • Se il gioco esce al menu HOME, l’exploit è stato installato correttamente
    9. Avvia nuovamente “Cubic Ninja”
    10. Se l’exploit ha avuto successo, la tua console avrà avviato l’Homebrew Launcher
    11. Spegni la tua console

    Sezione III - Verifica Pulsanti Hardware

    In questa sezione controllerai se la tua console può accedere alla Modalità Provvisoria. Serve per determinare quale metodo seguirai nella prossima pagina.

    In questa sezione verificherai se i pulsanti dorsali della tua console funzionano. Serve per determinare quale metodo seguirai nella prossima pagina.

    1. Accendi la tua console
    2. Nel menu HOME premi contemporaneamente i pulsanti L e R
      • L’applicazione fotocamera dovrebbe apparire
    3. Spegni la tua console

    Passaggio successivo: Scelta di un exploit

    Se è apparsa la fotocamera nella sezione precedente, la tua console ha la Modalità Provvisoria probabilmente funzionante.

    Se è apparsa la fotocamera, prosegui con Installazione di boot9strap (HBL-USM)

    Se la fotocamera NON è apparsa, prosegui con Dump di Movable (nimhax)
