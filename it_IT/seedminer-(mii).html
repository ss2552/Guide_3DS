    Seedminer (Mii)

    Se hai bisogno di aiuto, chiedi pure su Nintendo Homebrew su Discord (in inglese).
    Se hai gradito questa guida, accettiamo Donazioni.

    Indice

    • Istruzioni
      • Cosa serve
      • Sezione I - Preparazione
      • Sezione II - Elaborazione del QR Mii e Mining

    Questo metodo richiede una GPU ragionevolmente potente (una qualsiasi scheda grafica dedicata degli ultimi anni) e la possibilità di utilizzare una riga di comando. Se non sai cosa sono o non hai l’hardware necessario, unisciti al canale Nintendo Homebrew su Discord, apri il canale “#mii-mining-assistance” e fornisci quanto segue:

    • Il tuo codice QR Mii, caricato direttamente su Discord (non scattare uno screenshot o una foto dell’immagine, basta inviare l’immagine HNI_XXXX.JPG stessa)
    • ID0 della tua console (il nome della cartella a 32 caratteri dentro la cartella Nintendo 3DS - consulta questa sezione se ne hai più di uno)
    • Il modello della tua console (ad es. 3DS, 3DS XL, 2DS, New3DS, New3DS XL, New2DS XL)
    • L’anno di produzione della tua console; se non lo sai, non indicarlo - non è molto importante
    Dettagli Tecnici (opzionale)

    Per installare boot9strap nella tua console, ne ricaviamo la sua univoca chiave di cifratura. Utilizzeremo perciò un’applicazione chiamata Seedminer per calcolare la chiave di cifratura dati (movable.sed) per la tua console.

    Poiché non abbiamo accesso alla lista di amici, ricaviamo la prima parte del file movable.sed tramite il codice QR Mii anziché tramite lo scambio reciproco di amicizia.

    Per ulteriori informazioni sul funzionamento di Seedminer, è possibile consultare questa presentazione.

    Istruzioni

    Se sei su Linux o macOS, sostituisci py -3 con python3 in tutti i comandi.

    Cosa serve

    • L’ultima versione di Seedminer (il file .zip di Seedminer)
    • Qualsiasi versione 3.x di Python installata
      • Se sei su Linux, probabilmente hai già Python 3. Controlla aprendo un terminale e inserendo python3 -V. Se il comando restituisce un numero di versione, funzionerà per questa guida.

    Sezione I - Preparazione

    In questa sezione imposterai Seedminer e otterrai i file della tua console necessari per il mining.

    1. Estrai il contenuto dell’archivio .zip di Seedminer e sposta la cartella con il nome del tuo sistema operativo in qualunque posizione sul tuo PC
      • Mantieni questo file a portata di mano, dato che la cartella Seedminer verrà usata per posizionare altri file
    2. Nella cartella Seedminer, tieni premuto Maiusc e fai clic con il tasto destro del mouse in un qualunque punto vuoto della cartella, quindi fai clic sull’opzione che apre il terminale, PowerShell o il prompt dei comandi
    3. Nella finestra a riga di comando, digita py -3 -m pip install pycryptodomex, premi INVIO, dopodiché consenti l’installazione
      • Se il comando restituisce l’errore No module named pip, devi installare pip manualmente. Troverai le relative istruzioni in questa pagina.
    4. Accendi la console con la scheda SD inserita
    5. Apri l’applicazione Centro di creazione Mii
    6. Tocca il pulsante “QR Code e immagini”
    7. Tocca “Salva QR Code”
    8. Seleziona un Mii creato su questa console, NON importato da un’altra console (creane uno nuovo se non ne hai uno) e premi “OK”
    9. Spegni la tua console
    10. Inserisci la scheda SD nel tuo computer
    11. Entra nella cartella Nintendo 3DS della tua scheda SD
    12. Copia il nome della cartella a 32 lettere che vedi direttamente all’interno della cartella Nintendo 3DS
      • Questo nome a 32 lettere è specifico per il tuo sistema e sarà diverso per ogni console
      • Se vedi più cartelle a 32 lettere, segui queste istruzioni
      • Se presente, puoi ignorare la cartella private

    13. Incolla il nome della cartella a 32 lettere in un documento a cui fare riferimento più tardi
      • Il nome di questa cartella è conosciuto anche come il tuo “ID0”. Ricordatene perché questa guida ne farà riferimento come tale successivamente
    14. Entra nella cartella DCIM della scheda SD
    15. Cerca il codice QR, che sarà un file .JPG che comincia con HNI
      • Potresti trovare altri file se hai già utilizzato l’applicazione Fotocamera Nintendo 3DS
    16. Copia questo file nella cartella Seedminer sul tuo PC

    Sezione II - Elaborazione del QR Mii e Mining

    In questa sezione trasformerai il codice QR Mii in un file utilizzabile e con esso minerai il file movable.sed.

    1. Apri il sito editMii sul tuo computer
    2. Clicca su “Import from…”, quindi seleziona “QR Code” e carica il codice QR Mii ottenuto prima
    3. Clicca su “Export to…”, quindi seleziona “encrypted (.bin)”
      • Verrà scaricato un file chiamato input.bin
    4. Copia questo file input.bin nella cartella Seedminer
    5. Apri la finestra del terminale/PowerShell/Prompt dei comandi di prima
    6. In questa finestra, inserisci il comando py -3 seedminer_launcher3.py id0 <id0>
      • Sostituisci <id0> con l’ID0 che hai copiato precedentemente nel documento
      • Non includere le parentesi angolate
      • Non tentare di inserire l’ID0 manualmente, poiché è facile commettere un errore. Assicurati che l’ID0 sia inserito correttamente copiandolo e incollandolo da dove lo hai salvato nella sezione precedente
    7. Al termine, inserisci il comando py -3 seedminer_launcher3.py mii <modello> <anno>
      • Sostituisci <modello> con il modello della tua console (new o old) - se non ne sei sicuro, solo i modelli con i pulsanti ZL/ZR e C-Stick sono “new”
      • Sostituisci <year> con l’anno in cui è stato realizzato il 3DS (sulle console Old 3DS, gli anni sulla parte posteriore, (M)2010/(C)2011, sono sempre gli stessi e NON sono un indicatore usabile) - se non sei sicuro, vedi qui sotto
      • Gli anni di produzione approssimativi sono: Old 3DS 2011, Old 3DS XL 2013, Old 2DS 2015, New 3DS 2016, New 3DS XL 2016, New 2DS XL 2017
      • Puoi anche lasciare vuoto l’anno, per cui partirà dal mezzo
      • Il processo di mining richiede solitamente fino a 90 minuti, a seconda della fortuna e della potenza della tua GPU
      • La prima fase terminerà probabilmente attorno l’offset 1500, a seconda del modello di console
      • Se la seconda fase (dopo il messaggio movable_part2.sed generation success) oltrepassa l’offset 100, verifica di aver inserito l’ID0 corretto. Se il tuo ID0 è corretto, non puoi usare questo metodo e dovrai aggiornare all’ultima versione e controllare se è disponibile per te un metodo non-Seedminer qui
      • Se vedi il messaggio input.bin is invalid size, ricrea il file input.bin utilizzando il tuo codice QR. Se fallisce di nuovo, apri un browser diverso e ricrea il file input.bin
      • Se vedi il messaggio LFCS has been left blank, usa il modello di console opposto (ad es. mii old al posto di mii new)
    8. Al termine, apparirà un file movable.sed nella cartella Seedminer
