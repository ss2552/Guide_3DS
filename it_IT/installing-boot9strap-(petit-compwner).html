    Installazione di boot9strap (petit-compwner)

    Se hai bisogno di aiuto, chiedi pure su Nintendo Homebrew su Discord (in inglese).
    Se hai gradito questa guida, accettiamo Donazioni.

    Indice

    • Cosa serve
      • Sezione I - Controllo del CFW
      • Sezione II - Preparazione
      • Sezione III - petit-compwner
      • Sezione IV - Installazione di boot9strap
      • Sezione V - Configurazione di Luma3DS

    Dettagli Tecnici (opzionale)

    Useremo petit-compwner per avviare b9sTool, che installerà boot9strap.

    Vedi qui per maggiori informazioni su petit-compwner.

    Dovresti essere arrivato a questa pagina solo se esegui la versione di sistema 11.17.0. Se hai una versione del software diversa da 11.17.0 o 11.16.0, FERMATI SUBITO poiché queste istruzioni BRICKERANNO LA TUA CONSOLE con altri firmware!!

    Cosa serve

    • Utenti con 11.16.0 o 11.17.0: La versione v6.1.1 di b9sTool
      • Assicurati assolutamente di avere una di queste versioni del software, in quanto qualunque altra versione causerà un BRICK
    • L’ultima versione di Luma3DS (il file .zip di Luma3DS)

    Sezione I - Controllo del CFW

    Come precauzione aggiuntiva, controlleremo anche la presenza di un custom firmware. È necessario poiché se un custom firmware fosse già installato, questo metodo rischierebbe di brickare la console (rendendola inutilizzabile in assenza di metodi di ripristino come ntrboot).

    1. Spegni la tua console
    2. Premi il pulsante (Select)
    3. Accendi la tua console continuando a tenere premuto il pulsante (Select)
    4. Se il controllo è andato a buon fine, verrà avviato il menu HOME e potrai procedere con questa guida
    5. Spegni la tua console

    Se vedi un menu di configurazione o la console si spegne, hai già il CFW, e proseguire seguendo queste istruzioni può BRICKARE la console! Procedi con la Verifica del CFW per aggiornare il tuo CFW esistente.

    Sezione II - Preparazione

    In questa sezione copierai i file necessari per avviare b9sTool, che viene utilizzato per avviare il programma di installazione di boot9strap (custom firmware).

    1. Inserisci la scheda SD nel tuo computer
    2. Copia il file boot.nds (B9STool) dall’archivio .zip di b9sTool nella directory principale della tua scheda SD
      • La directory principale della scheda SD è la cartella della tua scheda SD in cui è possibile visualizzare la cartella Nintendo 3DS, ma non il suo interno
    3. Copia i file boot.firm e boot.3dsx dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
    4. Reinserisci la scheda SD nella tua console

    5. Accendi la tua console

    Sezione III - petit-compwner

    In questa sezione scansionerai i codici QR necessari per attivare l’exploit petit-compwner.

    1. Apri l’ultima versione di petit-compwner sul tuo computer
    2. Sulla tua console, avvia Petit Computer
    3. Seleziona File Management
    4. Seleziona Read QR Codes
    5. Scansiona ogni codice QR in ordine, da 1/5 a 5/5
    6. Una volta scansionati tutti i codici QR, esci da File Management
    7. Seleziona View Gallery
    8. Avvia il programma PWN
    9. Se l’exploit ha funzionato, la tua console avvierà b9sTool

    Sezione IV - Installazione di boot9strap

    In questa sezione installerai il custom firmware sulla tua console.

    1. Usando il D-Pad, vai su “Install boot9strap”
      • Se salti questo passaggio, la console si riavvierà nel menu HOME senza aver installato boot9strap, e sarà necessario ricominciare dalla sezione precedente
    2. Premi (A), quindi premi START e SELECT contemporaneamente per avviare il processo
    3. Una volta finito e nello schermo inferiore c’è scritto “done.”, esci da b9sTool e spegni la console
      • Potresti dover forzare lo spegnimento tenendo premuto il pulsante di accensione
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
      • Se vedi la schermata di Luma, spegni la console e prosegui alla prossima sezione

    Sezione V - Configurazione di Luma3DS

    1. Premi (Select) e accendi la tua console continuando a tenerlo premuto
    2. La tua console dovrebbe aver avviato il menu di configurazione di Luma3DS
      • Il menu di configurazione di Luma3DS contiene le impostazioni per il custom firmware Luma3DS. Molte di queste impostazioni possono essere utili per la personalizzazione o per debug
      • Ai fini di questa guida, lascia queste opzioni nelle impostazioni predefinite (non spuntare o deselezionare nulla)
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
    3. Premi (Start) per salvare e riavviare il 3DS

    A questo punto, la console avvierà sempre Luma3DS.

    • Luma3DS non è diverso dal normale menu HOME. Se la console avvia il menu HOME, vuol dire che sta eseguendo il custom firmware.
    • Nella pagina successiva installerai applicazioni homebrew utili per completare la tua installazione.
