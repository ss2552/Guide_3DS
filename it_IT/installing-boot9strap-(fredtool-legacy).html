    Installazione di boot9strap (Fredtool, Legacy)

    Se hai bisogno di aiuto, chiedi pure su Nintendo Homebrew su Discord (in inglese).
    Se hai gradito questa guida, accettiamo Donazioni.

    Dettagli Tecnici (opzionale)

    Per eseguire un dump dei DSiWare di sistema, approfittiamo di una falla di sicurezza nella sezione Gestione Nintendo DSiWare dell’applicazione Impostazioni della console.

    Per realizzare l’obiettivo, utilizziamo la chiave di cifratura della tua console (movable.sed) per generare un backup DSiWare che induce la console a creare un dump dell’applicazione Impostazioni Internet DSi nella directory principale della scheda SD.

    Una volta ottenuto un backup DSiWare, possiamo iniettarlo nell’applicazione Connessioni Nintendo DS.

    Questa è un’implementazione funzionante dell’exploit del “testo in chiaro noto nelle partizioni FIRM” spiegato qui.

    Dovresti essere arrivato a questa pagina solo se esegui la versione 11.15.0 o 11.14.0. Se hai una versione del software diversa da 11.15.0 o 11.14.0, FERMATI SUBITO poiché queste istruzioni BRICKERANNO LA TUA CONSOLE con altri firmware!!

    Cosa serve

    • Il tuo file movable.sed dalla sezione Seedminer (Mii)
    • L’ultima versione di Frogminer_save (Frogminer_save.zip)
    • Solo utenti con 11.15.0 o 11.14.0: La versione v6.0.1 di b9sTool (download diretto)
    • L’ultima versione di Luma3DS (il file .zip di Luma3DS)

    Sezione I - Controllo del CFW

    Come precauzione aggiuntiva, controlleremo anche la presenza di un custom firmware. È necessario poiché se un custom firmware fosse già installato, questo metodo rischierebbe di brickare la console (rendendola inutilizzabile in assenza di metodi di ripristino come ntrboot).

    1. Spegni la tua console
    2. Premi il pulsante (Select)
    3. Accendi la tua console continuando a tenere premuto il pulsante (Select)
    4. Se il controllo è andato a buon fine, verrà avviato il menu HOME e potrai procedere con questa guida
    5. Spegni la tua console

    Se vedi un menu di configurazione o la console si spegne, hai già il CFW, e proseguire seguendo queste istruzioni può BRICKARE la console! Procedi con la Verifica del CFW per aggiornare il tuo CFW esistente.

    Sezione II - BannerBomb3

    In questa sezione attiverai l’exploit BannerBomb3 utilizzando il menu Gestione Nintendo DSiWare e copierai il file dump risultante sul computer per poterlo utilizzare nella sezione successiva.

    1. Reinserisci la scheda SD nella tua console
    2. Accendi la tua console
    3. Avvia le Impostazioni di sistema sulla tua console
    4. Entra in Gestione dati -> DSiWare -> Scheda SD (image)
      • La tua console dovrebbe mostrare il menu BB3 multihax
      • Se questo passaggio causa il crash della tua console, segui questa guida per risolvere il problema
    5. Usa il D-Pad per spostarti sul menu “Dump DSiWare”, quindi premi il pulsante (A)
      • La tua console si riavvierà automaticamente
    6. Spegni la tua console

    Sezione III - Preparazione

    In questa sezione copierai i file necessari per sostituire temporaneamente “Impostazioni di collegamento” con Flipnote Studio, usato per lanciare l’installer di boot9strap (custom firmware).

    1. Inserisci la scheda SD nel tuo computer
    2. Apri il sito DSIHaxInjector_new sul tuo computer
    3. Sotto il campo “Username” inserisci un nome alfanumerico qualunque (senza spazi o caratteri speciali)
    4. Sul il campo “DSiBin”, carica il tuo file backup DSiWare (ad esempio 42383841.bin) presente nella directory principale della tua scheda SD usando la prima opzione “Browse…”
    5. Sul campo “MovableSed”, carica il tuo file movable.sed usando la seconda opzione “Browse…”
    6. Sul campo “InjectionTarget”, imposta l’obiettivo dell’iniezione su DSinternet (NON memorypit)
    7. Clicca “Build”
      • Attendi qualche secondo che termini il processo di compilazione
    8. Nella sezione “Build History” a sinistra, inserisci l’Username nel campo “Filter Builds”
    9. Clicca sul primo risultato della ricerca
      • Dovrebbe avere la data più recente
    10. Clicca il link “output_(name).zip”
    11. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
      • <ID0> è il nome a 32 lettere della cartella che hai copiato in Seedminer
      • <ID1> è una cartella dal nome lungo 32 caratteri all’interno di <ID0>
    12. Elimina il file F0D43D5.bin dalla cartella Nintendo DSiWare
    13. Copia il file 42383841.bin dalla cartella hax dall’archivio DSiWare appena scaricato (output_(name).zip) nella cartella Nintendo DSiWare
    14. Copia i file boot.firm e boot.3dsx dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
      • La directory principale della scheda SD è la cartella della tua scheda SD in cui è possibile visualizzare la cartella Nintendo 3DS, ma non il suo interno
    15. Copia il file boot.nds (B9STool) dall’archivio .zip di b9sTool nella directory principale della tua scheda SD
    16. Copia la cartella private presente nell’archivio .zip di Frogminer_save nella directory principale della tua scheda SD
    17. Reinserisci la scheda SD nella tua console

    18. Accendi la tua console

    Sezione IV - Sovrascrittura delle Impostazioni di collegamento DS

    In questa sezione copierai il DSiWare modificato “Impostazioni di collegamento DS” nella memoria interna, che verrà così temporaneamente sostituito da Flipnote Studio.

    1. Avvia le Impostazioni di sistema sulla tua console
    2. Entra in Gestione dati -> DSiWare -> Scheda SD (image)
    3. Seleziona “Haxxxxxxxxx!”
      • Se non riesci a selezionare “Haxxxxxxxxx”, segui questa guida per risolvere il problema
    4. Seleziona “Copia”, quindi “OK”
    5. Ritorna al menu principale delle Impostazioni della console
    6. Entra in Impostazioni Internet -> Connessioni Nintendo DS, quindi seleziona “OK” (immagine)
    7. Se l’exploit ha avuto successo, la tua console avrà caricato la versione JPN di Flipnote Studio
      • Se la tua console non carica la versione JPN di Flipnote Studio, segui questa guida per risolvere il problema

    Sezione V - Exploit Flipnote

    Se desideri una guida visiva per questa parte, è disponibile qui.

    In questa sezione, eseguirai una serie passaggi molto specifici da fare in Flipnote Studio che, se fatti correttamente, farà partire b9sTool, l’installer di boot9strap (custom firmware).

    1. Completa la configurazione iniziale del gioco finché non avrai raggiunto il menu principale
      • Seleziona l’opzione di sinistra quando richiesto durante la configurazione
      • Se riscontri un problema durante questa sezione, cerca in questa guida il tuo problema
    2. Con il touch screen, seleziona il riquadro grande a sinistra, poi il riquadro con l’icona di una scheda SD
    3. Al termine del caricamento del menu, scegli l’icona col volto, poi premi l’icona in basso a destra per proseguire
    4. Premi (X) o (SU) sul D-Pad a seconda di cosa sia visualizzato sullo schermo superiore
    5. Seleziona il secondo pulsante in cima, con l’icona di una pellicola
    6. Scorri a destra fino a visualizzare la bobina “3/3”
    7. Seleziona il terzo riquadro con la lettera “A” all’interno
    8. Scorri a sinistra fino a visualizzare la bobina “1/3”
    9. Seleziona il quarto riquadro con la lettera “A” all’interno
    10. Se l’exploit ha funzionato, la tua console avvierà b9sTool
    11. Usando il D-Pad, vai su “Install boot9strap”
      • Se salti questo passaggio, la console si riavvierà nel menu HOME senza aver installato boot9strap, e sarà necessario riaprire l’applicazione Connessioni Nintendo DS e ricominciare dall’inizio di questa sezione
    12. Premi (A), quindi premi START e SELECT contemporaneamente per avviare il processo
    13. Una volta finito e nello schermo inferiore c’è scritto “done.”, esci da b9sTool e spegni la console
      • Potresti dover forzare lo spegnimento tenendo premuto il pulsante di accensione
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
      • Se vedi la schermata di Luma, spegni la console e prosegui alla prossima sezione

    Sezione VI - Configurazione di Luma3DS

    1. Premi (Select) e accendi la tua console continuando a tenerlo premuto
    2. La tua console dovrebbe aver avviato il menu di configurazione di Luma3DS
      • Il menu di configurazione di Luma3DS contiene le impostazioni per il custom firmware Luma3DS. Molte di queste impostazioni possono essere utili per la personalizzazione o per debug
      • Ai fini di questa guida, lascia queste opzioni nelle impostazioni predefinite (non spuntare o deselezionare nulla)
      • Se la tua console si spegne quando cerchi di accenderla, assicurati di aver copiato il file boot.firm dall’archivio .zip di Luma3DS nella directory principale della tua scheda SD
    3. Premi (Start) per salvare e riavviare il 3DS

    A questo punto, la console avvierà sempre Luma3DS.

    • Luma3DS non è diverso dal normale menu HOME. Se la console avvia il menu HOME, vuol dire che sta eseguendo il custom firmware.
    • Nella pagina successiva installerai applicazioni homebrew utili per completare la tua installazione.

    Sezione VII - Ripristino delle Impostazioni di Connessione DS

    In questa sezione ripristinerai “Impostazioni di collegamento DS” nel modo in cui era prima che fosse temporaneamente sostituito con Flipnote Studio in una sezione precedente.

    1. Nella tua scheda SD, entra in Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare
    2. Copia il file 42383841.bin dalla cartella clean dall’archivio DSiWare appena scaricato (output_(name).zip) nella cartella Nintendo DSiWare, sostituendo quello preesistente
    3. Reinserisci la scheda SD nella tua console
    4. Accendi la tua console
    5. Avvia le Impostazioni di sistema sulla tua console
    6. Entra in Gestione dati -> DSiWare -> Scheda SD (image)
    7. Seleziona “Nintendo DSi™”
    8. Seleziona “Copia”, quindi “OK”
