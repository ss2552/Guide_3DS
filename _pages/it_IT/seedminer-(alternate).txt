### Passi successivi

Seleziona il metodo in base all'applicazione che hai installato sul tuo 3DS. Non importa il metodo che sceglierai, il risultato finale sarà lo stesso.

#### Mario Kart 7

Questo metodo utilizza una scheda di gioco o una copia digitale del gioco Mario Kart 7. Using the `movable.sed` file, you can modify Mario Kart 7's SD card data that will give you access to Homebrew Launcher, which will be used for further exploitation.

This method requires a computer running Windows.
{: .notice--warning}

Continue to [Installing boot9strap (Kartminer7)](installing-boot9strap-(kartminer7))
{: .notice--primary}

#### DSiWare

Se possiedi un qualunque [titolo DSiWare](https://en.wikipedia. rg/wiki/List_of_DSiWare_games_and_applications) sul tuo 3DS, è possibile scaricarlo sulla scheda SD e utilizzarlo per iniettare temporaneamente Flipnote nelle Impostazioni di collegamento DS.

Prosegui con [Dump DSiWare eShop](dumping-eshop-dsiware)
{: .notice--primary}

#### PicHaxx

Questo metodo utilizza il gioco Pokémon Picross, un gioco che era gratuito. Utilizzando il file `movable.sed` puoi creare un file di salvataggio modificato che ti darà accesso all'Homebrew Launcher, che verrà usato per exploit successivi.

Prosegui con l'[Installazione di boot9strap (PicHaxx)](installing-boot9strap-(pichaxx))
{: .notice--primary}

#### steelhax

Questo metodo utilizza il gioco Steel Diver: Sub Wars, un gioco che era gratuito. Utilizzando il file `movable.sed` puoi creare un file di salvataggio modificato che ti darà accesso all'Homebrew Launcher, che verrà usato per exploit successivi.

Prosegui con [Installazione di boot9strap (Steelhax)](installing-boot9strap-(steelhax))
{: .notice--primary}
