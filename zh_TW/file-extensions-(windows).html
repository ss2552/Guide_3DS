    檔案副檔名 (Windows)

    若需英語支援，請於 Discord 上的 Nintendo Homebrew 伺服器發問。

    必讀事項

    本篇為改變預設 Windows 隱藏副檔名行為的附加章節。

    顯示副檔名可以讓您在找本教學提到的檔案時更方便些。

    操作說明

    Windows 11

    1. 於任何資料夾中開啟『檔案總管』
    2. 點選上方的『檢視』按鈕
    3. 選擇『顯示』

    4. 點選『檔案名稱附檔名』

    Windows 10 / 8.1 / 8

    1. 於任何資料夾中開啟『檔案總管』
    2. 點選上方的『檢視』按鈕

    3. 將『副檔名』格子打勾

    Windows 7

    1. 透過左下角開始按鈕或按下鍵盤上的 Windows 鍵以開啟開始選單

    2. 尋找『資料夾選項』

    3. 於資料夾選項選單中上方點下『檢視』籤頁

    4. 將『隱藏已知檔案類型的副檔名』選項取消
