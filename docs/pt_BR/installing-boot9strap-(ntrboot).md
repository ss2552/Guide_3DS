# Instalando boot9strap (ntrboot)

## Required Reading

Para usar os links do [magnet](https://wikipedia.org/wiki/Magnet_URI_scheme) nesta página, você precisará de um cliente de torrent como [qBittorrent](https://www.qbittorrent.org/download.php) ou [Deluge](http://dev.deluge-torrent.org/wiki/Download).

## What You Need

- A magnet that triggers the sleep mode of your console (if using a folding style console)
- Your ntrboot flashed flashcart
- The latest release of [SafeB9SInstaller](https://github.com/d0k3/SafeB9SInstaller/releases/download/v0.0.7/SafeB9SInstaller-20170605-122940.zip) (direct download)
- The latest release of [boot9strap](https://github.com/SciresM/boot9strap/releases/download/1.4/boot9strap-1.4.zip) (direct download)
- The latest release of [Luma3DS](https://github.com/LumaTeam/Luma3DS/releases/latest) (the Luma3DS `.zip` file)

## Instructions

### Section I - Prep Work

1. Desligue seu console

2. Insira o cartão de SD do seu dispositivo no seu computador
   - This is the SD card from your 3DS, _not_ the SD card from your flashcart

3. Copie `SafeB9SInstaller.firm` do `.zip` do SafeB9SInstaller para a raiz do seu cartão SD e renomeie-o para `boot.firm`
   - The root of the SD card refers to the initial directory on your SD card where you can see the Nintendo 3DS folder, but are not inside of it

4. Crie uma pasta chamada `boot9strap` na raiz do seu cartão SD

5. Copie o `boot9strap.firm` e o `boot9strap.firm.sha` do `.zip` do boot9strap para a pasta `/boot9strap/` no seu cartão SD

   ::: info

   ![](/images/screenshots/boot9strap-ntrboot-file-layout.png)

   :::

6. Reinsira o cartão SD no seu console

7. Ligue o seu console

### Section II - ntrboot

1. Use o ímã para encontrar o local no seu console onde o sensor de descanso é acionado
   - This step is not needed on the old 2DS (which has a sleep mode switch)
2. Desligue seu console
3. Insira o seu flashcart no seu console
4. Coloque o ímã no seu console para ativar o sensor de descanso
   - On old 2DS, you should instead enable the sleep mode switch
5. Segure os botões (Start) + (Select) + (X) + (Power) por muitos segundos, então solte os botões
   - It may take a few attempts to get this to work because the positioning is awkward
6. Se o exploit for bem-sucedido, você terá entrado no SafeB9SInstaller
7. Remova o ímã do seu console
   - On old 2DS, you should instead disable the sleep mode switch

### Section III - Installing boot9strap

Nesta seção, você instalará custom firmware no seu console.

1. Quando solicitado, aperte a sequência de botões fornecida na tela superior para instalar o boot9strap
   - If a step on the lower screen has red-colored text, and you are not prompted to input a key combo, [follow this troubleshooting guide](troubleshooting#issues-with-safeb9sinstaller)
2. Assim que for completo, force o seu console a desligar segurando o botão POWER
   - Your console will only boot to the SafeB9SInstaller screen until the next section is completed

### Section IV - Configuring Luma3DS

1. Insira o cartão de SD do seu dispositivo no seu computador
2. Delete o `boot.firm` da raiz do seu cartão SD
3. Copie `boot.firm` e `boot.3dsx` do `.zip` do Luma3DS para a raiz do seu cartão SD
4. Reinsira o cartão SD no seu console
5. Ligue o seu console

Observe que o Acekard 2i mantém sua capacidade de executar arquivos '.nds' tendo o exploit do ntrboot instalado. Isso só se aplica quando o Acekard 2i esta em um 3DS executando algum tipo de custom firmware! Enquanto o ntrboot exploit é instalado no Acekard 2i, não será possível iniciar arquivos `.nds` nos sistemas DSi ou 3DS de varejo.

Não siga esta seção até completar o resto das instruções nesta página.

- The latest release of [ntrboot_flasher](https://github.com/ntrteam/ntrboot_flasher/releases/latest) (`ntrboot_flasher.firm`)

  - Note that if you followed [Flashing ntrboot (3DS Multi System)](flashing-ntrboot-\(3ds-multi-system\)), the flashrom backup already exists in the correct location and does not need to be downloaded

  - If you do not know which HW revision you have, just try each for your cart of them. Somente o correto permitirá que seu flashcart seja executado corretamente a partir do Menu HOME, mas executar o flash errado não irá causar um brick no flashcart

#### Instructions

1. Desligue seu console
2. Insira o cartão SD no seu computador
   - This is the SD card from your 3DS, _not_ the SD card from your flashcart
3. Crie uma pasta chamada `ntrboot` na raiz do seu cartão SD
4. Copie o arquivo `bin` do seu backup da flashrom `.zip` para a pasta `/ntrboot/` na raíz de seu cartão SD
5. Crie uma pasta chamada `payloads` na pasta 'luma' do seu cartão SD
6. Copie `ntrboot_flasher.firm` para a pasta `/luma/payloads/` no seu cartão SD
7. Reinsira o cartão SD no seu console
8. Insira o seu flashcart DS / DSi compatível com ntrboot no seu console
9. Inicie o ntrboot_flasher segurando (Start) durante a inicialização
10. Leia o aviso em vermelho na tela
11. Aperte (A) para continuar
12. Selecione seu flashcart
    - If you do not see your flashcart in the list at the top, read the bottom screen for more info on each option
13. Selecione "Restore Flash"
14. Aperte (A) para continuar
15. Aguarde até que o processo seja concluído
16. Aperte (A) para retornar ao menu principal
17. Aperte (B) para desligar seu console
