# boot9strap 설치 (MSET9)

## 운영체제 선택

MSET9는 **콘솔 해킹에 사용하는** 컴퓨터, 스마트폰 또는 태블릿에 따라 절차가 다릅니다.

**콘솔 해킹에 사용할** 운영 체제를 선택해 주세요. 이 기기는 3DS SD 카드에 파일을 읽거나 저장할수 있어야 합니다.
