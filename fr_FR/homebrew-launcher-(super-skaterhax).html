    Homebrew Launcher (super-skaterhax)

    Pour un support en Anglais, demander de l'aide sur le Discord Nintendo Homebrew.

    Notes de compatibilité

    Super-skaterhax (when used to launch the Homebrew Launcher) is compatible with New models on version 11.16.0 in all regions, and 11.17.0 in the EUR and JPN regions.

    Les instructions fournies ne s’appliquent qu’aux New 3DS, New 3DS XL et New 2DS XL. Veuillez vous assurer que la console que vous hackez est une New 3DS, New 3DS XL ou New 2DS XL avant de continuer.

    Ce dont vous avez besoin

    • La dernière version de Luma3DS (le fichier `.zip’ de Luma3DS)
    • La dernière version de super-skaterhax (le fichier .zip)

    Section I - Préparatifs

    Dans cette section, vous copierez les fichiers nécessaires pour déclencher à la fois super-skaterhax et l’Homebrew Launcher.

    1. Éteignez votre console
    2. Insérez votre carte SD dans votre ordinateur
    3. Copiez boot.firm et boot.3dsx depuis Luma3DS .zipvers la racine de votre carte SD
      • La racine de la carte SD représente le répertoire initial de la carte SD où vous pouvez voir le dossier Nintendo 3DS, mais ce n’est pas à l’intérieur de ce dernier
    4. Copiez tout ce qui se trouve dans le dossier correspondant à la région de votre console dans le fichier .zip à la racine de votre carte SD
    5. Retirez la carte SD de votre ordinateur et réinsérez-la dans votre console
    6. Allumez votre console

    Section II - super-skaterhax

    Dans cette section, vous allez visiter la page web d’exploit du Navigateur Internet, qui lancera l’Homebrew Launcher.

    Si vous ne l’avez pas déjà fait, assurez-vous d’avoir une connexion Internet fonctionnelle configurée sur votre console.

    1. Sur le Menu HOME, appuyez simultanément sur les boutons (L) et (R) pour ouvrir l’appareil photo
      • Si vous ne pouvez pas ouvrir la caméra, ouvrez le Navigateur Internet et tapez manuellement l’URL (https://zoogie.github.io/web/super/ pour EUR/USA/JPN, https://zoogie.github.io/web/korea pour KOR)
    2. Appuyez sur le bouton QR code et scannez l’un des codes QR suivants pour la région de votre console ici
      • Si vous recevez un avertissement de certificat de sécurité, appuyez sur (A) pour autoriser la connexion
    3. Appuyez sur le bouton (Select) pour ouvrir l’onglet signet
      • Si votre bouton (Select) ne répond pas, appuyez sur l’icône étoile en bas à gauche de l’écran
    4. Appuyez sur “Ajouter cette page aux favoris”
    5. Appuyez une fois sur (B) pour revenir au navigateur
    6. Appuyez sur (Start) pour ouvrir le menu contextuel
      • Si votre bouton (Start) ne répond pas, appuyez sur le bouton avec 3 lignes en bas à droite de l’écran
    7. Appuyez sur “Paramètres”
    8. Appuyez sur “Effacer les cookies”
    9. Appuyez sur (A) pour continuer
    10. Appuyez sur (HOME) pour revenir au Menu HOME, puis appuyez immédiatement sur (A) pour relancer le navigateur
    11. Appuyez sur le bouton “GO GO!” en haut de l’écran
      • Si des invites apparaissent, approuvez les tous
      • Si votre console se bloque sur un écran jaune, maintenez le bouton POWER jusqu’à ce qu’elle s’éteigne, puis réessayez cette section
      • If your device freezes on a red screen, hold the POWER button until it turns off, redo step 3 of Section II, then retry this section
      • If you get another error, try again up to 5 times, and if it still doesn’t work, follow this troubleshooting guide
    12. Si l’exploit a réussi, votre console aura démarré dans l’Homebrew Launcher
    13. Éteignez votre console

    Section III - Hardware Button Check

    Dans cette section, vous verrez si les boutons (L) et (R) de votre console fonctionnent correctement. Cela déterminera la méthode que vous allez suivre sur la page suivante.

    1. Allumez votre console
    2. Une fois arrivé sur le Menu HOME, appuyez sur les boutons (L) et (R) en même temps
      • L’applet de l’appareil photo numérique devrait apparaître
    3. Éteignez votre console

    Étape suivante : Choisissez un exploit

    Si l’appareil photo numérique est apparu dans la section précédente, le mode sans échec devrait fonctionner sur votre console.

    If the camera appeared, continue to Installing boot9strap (HBL-USM)

    If the camera did NOT appear, continue to Dumping Movable (nimhax)
